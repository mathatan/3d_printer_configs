# 3d Printer configs

## Kansiorakenne

### star-g-code.txt

Sijoita tämä slicerin `before print g-code` tai `start g-code` kohtaan. Sisältää esilämmityksen, kotiuttamisen ja tulostuspään esilatauksen.

### end-g-code.txt

Sijoita tämä slicerin `after pring g-code` tai `end g-code` kohtaan. Sisältää tulostuspään tieltä siirron ja lämpöjen katkaisun.

### print_area_and_head.png

Kuva tulostus alueen ja tulostus pään koosta. Hyödyllinen etenkin jos haluat käyttää osakohtaista tulostus moodia. Helpottaa vähän asettelussa.

### materials

Perus materiaaliasetukset curaan. Orca-slicerissä voi mennä tulostimen defaulteilla: Generic PLA, PETG, TPU, ABS, jne.
